## 写在前面之这个项目是做什么的:

## 我们都需要些什么
        （什么都没有，你猜需要什么，都不知道做些什么呢！喵）
  1. App的整体风格与图标的选取，恶搞无下限是我们的追求（估计需要ps人才 + 阿里矢量图库）
  2. 天气类接口相关的查询与界面的设计。（乱七八糟的动画啦，折线图啦）

  4. 新闻接口的抓取和界面布局
  5. 估计要有图片上传把，七牛很好用 
  6. 还是希望技术上有所提高的，所以代码规范还是要的。
